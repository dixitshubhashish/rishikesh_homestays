# 📦 Deployment

Deploy in minutes with Netlify.

1. Fork or clone this repository.
2. Connect the repository to Netlify.
3. Publish directory:

```
.
```

4. Build command:

```
None
```

5. Add your custom domain

```
rishikeshhomestays.com
```

---
